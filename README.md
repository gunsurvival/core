## Getting Started

To get a local build, copy up and running follow these simple example steps.

### Prerequisites

- Install [node.js](https://nodejs.org/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/gunsurvival/core.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build the core
   ```sh
    npm run watch
   ```
